# Portafolio web v2.0 frontend



- #####[https://portafolio-gaston-m-frontend.herokuapp.com/](https://portafolio-gaston-m-frontend.herokuapp.com/) 



## Introducción:
- El proyecto de portafolio web es el trabajo final del bootcamp de Argentina Programa de duración de 7 meses en el cual plasmaremos todo lo aprendido sobre el desarrollo web fullstack el cual tiene como requisito ser desarrollado con las tecnologías de Angular, Spring boot y SQL. 


## Objetivos del programa:
- El objetivo del programa es crear una página web responsive para poder mostrar y modificar información del usuario con el objetivo de hacer visible sus habilidades, trayectoria, cursos , capacitaciones realizadas, educación , experiencia laboral, proyectos  creados e información de contacto .

## Funcionamiento del programa:

### Modos de funcionamiento:
##### La pagina web tiene 2 modos de funcionamiento:
- El modo espectador. El cual es la vista publica  de la pagina en la cual el espectador puede ver el contenido de la pagina y usar la navbar pero no tiene permitido editar la pagina

- El modo editor. Para ingresar al modo editor se tiene que ingresar un usuario y una password. Si son correctas la pagina entrara en modo editor la cual mostrara mas botones para poder agregar, editar o eliminar ítems. Si se realizazon todos los cambios podrá cerrar la sesión para desactivar el modo de edición

### Descripción de contenido de carpetas:

### Carpeta main

##### Angular.json
- Se agrego a styles: boostrap, PrimeNg, PrimeIcons, PrimeFlex y el tema lara        light blue para primeNg

- Se agrego a scrips: Popper, jquery y boostrap

### Carpeta src

##### App-routing.module.ts

- Continene todas las rutas creadas para el uso de angular routes

#### App.module.ts

- Contiene las importaciones de los modulos de angular, los componentes de angular y los componentes usados de NgPrime

#### Carpeta src /assets/

- Contiene las imágenes usadas para la creación de  pagina

#### Carpeta src /app/ ComponenteLogin

- Contiene los archivos correspondiente  al inicio de sesión en el cual se pedirá el usuario y la contraseña para iniciar el modo editor del programa

#### Carpeta src /app/componentes-port

- Esta carpeta con tiene otras carpetas que contienen los componentes que forman en su conjunto el proyecto

#### Carpeta src /app/componentes-port/certificados

- Esta carpeta contiene los archivos que corresponde a la parte de hard skills la cual esta compuesta por varios acordions (componente PrimeNg ) los cuales hacen de categoría para el tipo de certificado que contiene. Cada acordion en su interior contiene cards (componente PrimeNg) en las cuales se representan los certificados que fui obteniendo los cuales contienen un botón (componente PrineNg) que redirecciona a la pagina que valida el certificado. 

#### Carpeta src /app/componentes-port/experiencia

- Esta carpeta contiene los archivos que corresponde a la parte de experiencia y educacion la cual esta compuesta por 2 timeline (componente primeNG) en donde se muestra de forma vertical la experiencia laborl y la educación recibida 

#### Carpeta src /app/componentes-port/proyectos

- Esta carpeta contiene los archivos que corresponde a la parte de proyectos la cual consta de subcarpetas en la cual se contiene la descripción de los proyectos individualmente.
- En pantalla podemos ver que los proyectos se muestran en un carousel (componente PrimeNG) y al dar en el botón mas información despliega una ventana la cual mostrara la descripción del proyecto

#### Carpeta src /app/componentes-port/tarjeta-presentacion

- Esta carpeta contiene los archivos que corresponde a una breve presentación personal esta formada por una card (componente PrimeNg)

#### Carpeta src /app/componentes-port/componentesEdicion/(subcarpetas)

- Esta carpeta contiene todos los componentes que se cargan al momento de entrar en el modo edición y los cuales despliegan una ventana con los datos que se quieran editar 

#### Carpeta src /app/componentes-port/contacto

- Esta carpeta contiene los archivos que corresponde a la información de contacto con iconos de (prime icons) 

#### Carpeta src /app/componentes-port/home/cartel-inicio

- Esta carpeta contiene los archivos que forman la primera ventana de inicio que redirecciona al lobby principal

#### Carpeta src /app/componentes-port/jwt

- Esta carpeta contiene todos los archivos referentes al JWT dentro de sus subcarpetas se puede encontrar los modelos del usuario, login, jwt.
- En la subcarpeta service encontramos los servicios de autentificación interceptores y generadores de token 

####  Carpeta src /app/componentes-port/lobby
 
- En la carpeta lobby  encontramos encontramos la barra de navegación superior junto con el botón de inicio se sesión para el modo de edición. También desde este modulo se hace la invocación al resto de modulos que contiene el programa 

#### Carpeta src /app/componentes-port/modelos

- En la carpeta de modelos se encuentran agrupadas todas las interfaces que sirven para traer los datos desde el backend (springboot)

#### Carpeta src /app/componentes-port/service

- En la carpeta de service se encuentran agrupados todos los service que se encargan de la manipulación del CRUD de cada parte del programa. El cual trae sus datos de la base de tados SQL mediante el backend (springboot)
